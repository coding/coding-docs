以下流程图展示了 CODING DevOps 软件开发平台的基本操作流程。

1.  如要开始使用 CODING DevOps，您需要先创建或加入一个团队。
2.  加入团队之后，您可以在团队内创建项目或受邀加入别的项目。

3.  项目成员通过**项目协同**、**代码仓库**、**持续集成**、**持续部署**等功能实现项目协作。
